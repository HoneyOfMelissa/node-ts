# Node.ts

Node.js + TypeScript の環境を作成するテンプレート

## Getting Started

    # プロジェクト作成
    git clone git clone https://github.com/HoneyOfMelissa/node-ts.git {AnyName}
    cd {AnyName}
    rm -rf .git

    # パッケージインストール
    yarn install

    # サーバー起動
    yarn start

    # ブラウザからアクセス
    http://localhost:3000/
